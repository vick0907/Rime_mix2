# 小狼毫注音（洋蔥 mix-in 版）

## 內容說明：

- 由於日文、韓文、俄文、希臘文對我來說使用率太低，因此移除相關字典，減少連打誤判率。

## 使用方法：

- 安裝小狼毫輸入法本體
- 將Github內容覆蓋至以下路徑（Linux、Mac並無測試）

```
%APPDATA%\Rime  ( Windows 小狼毫 )

```

- 注音（洋蔥 mixin 版）鍵位

  > 四個衍伸方案：「1」標準版、「2」只有後綴易懂、「3」語言分野最明減少撞碼、「4」集中下排手順最好

  > 本人多使用「4」，因較喜歡鍵盤下排之手順，也推薦「3」
